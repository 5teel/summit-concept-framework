# Summit asset shelf

Before treating any idea as new, check it against this shelf. Reuse beats rebuild. If an existing asset covers ~80%, the concept becomes "extend X", which is a faster yes.

- **Store Explorer / Store Atlas** (McDonald's PH) — map-based store browser: pins, per-store detail, imagery, expansion/white-space mode with priority pins + SWOT. *Point here when: any "explore stores on a map", site selection, or network idea.*
- **SWOT Battle Cards** — per-store playbook cards, filterable, comparable. *When: per-store action plans or field-team packs.*
- **Closing Time (Late-Trade Finder)** — productised "should this store trade longer?" analysis. *When: trading-hours or daypart ideas.*
- **The Bump** — scrollytelling narrative report (7-Eleven). *When: story-first, scroll-driven insight pieces.*
- **Sales vs Price (SvP)** — price/volume decomposition suite; also the template for the multi-tenant modern-report platform. *When: pricing, promo mechanics, elasticity.*
- **Growth Breakdown / 711 Scatter** — additive growth decomposition (Stores × Transactions × IPT × Price sum to growth). *When: "why did sales move" ideas.*
- **Category Manager Report** (7-Eleven golden) — the weekly category workhorse. *When: category review cadence ideas.*
- **NPD PIR Scorecard** — monthly new-product post-implementation review. *When: NPD/launch tracking.*
- **New/Lost/Retained shopper engine** — native cohort-transition analysis on loyalty data. *When: lapsed shoppers, switching, retention.*
- **Deck Builder** — one-button VI report → PowerPoint. *When: "export this to slides" ideas.*
- **Ritchies storytelling engine** — VI report → generated PPT narrative. *When: automated client readouts.*
- **Distance Calculator** — CSV drag-drop proximity tool. *When: geo/proximity utilities.*
- **Report Atlas / Report Info Catalogue** — lineage + glossary of existing reports. *When: "what do we already have?"*
- **Campaign Dashboard** — campaign performance tracking. *When: media/activation measurement.*
- **Golden Reports** — 9 gold-standard benchmark report configs. *When: any new VI report — check if one is the pattern to copy.*

This list is a seed, not a census — when in doubt, ask the user whether something like this already exists on their account, and flag ≥80% overlaps as "extend, don't rebuild".
