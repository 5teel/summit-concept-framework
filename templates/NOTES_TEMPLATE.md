# Notes

Frictionless scratch buffer. Drop a line whenever a thought, choice, or dead-end
happens — no structure required, no commit ceremony. The harvest reads this plus
git history to propose CDRs, then clears the lines it used.

This is NOT a record. Nothing here is durable. If it matters, it becomes a CDR at
the next harvest. If it doesn't, it gets cleared. Never reference a NOTES line from
anywhere else — it won't exist for long.

## Optional one-char tags (use only if it's effortless)
- `!`  feels decided — a fork I chose and probably won't reverse  → harvest prioritises
- `?`  open question / unresolved                                  → harvest ignores until resolved
- (no tag) just an observation                                     → harvest weighs as context
