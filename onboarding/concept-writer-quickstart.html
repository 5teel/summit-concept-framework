Concept Writer — Quickstart

One-time setup, then two commands to Claude (Cowork or Code).

0
One-time setup. Double-click Setup-ConceptWriter.bat — maps X: and, for Claude Code, installs the plugin.
In Claude Cowork: Plugins panel → Add marketplace → 5teel/summit-concept-framework → install summit-concepts, then Add folder twice: X:\Labs and the workspace path the installer printed.
once
1
Point Claude at your concept
Registers it on the Concept Dashboard so the team can see it.
> summit-concept import "C:\path\to\your\project"
New idea instead of an existing build?  summit-concept my-idea
2
Build & iterate
Explore freely. Drop a one-line note when something feels decided.
> (build your POC — the concept stays visible on the dashboard)
3
Submit when it's demonstrable
A quick completeness check, then it's ready for a Strategist.
> summit-concept submit
✓
Done
It's on the Concept Dashboard. A Strategist takes it from here — you don't need to do anything else.
